## in-built functions

For a full list of the built in functions go to [built-in](https://www.programiz.com/python-programming/methods/built-in)

### print 

```
print("Hello World!") # Hello World!
`

### len

```
len("Hello World!") # 13 
`
### abs

```
abs(-17) # 17
`


### type

```
type(17.5) # float 
type("23") # string
`

### round

```
round(17.6) # 18
`
### max or min

```
  max(2,5,7,71,3) # 71
  
  min(-2,3,5,33) # -2
`
